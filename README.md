# Novel Phase Transition in Self-Driven Particles

This project implements a system of **Self-Driven Particles (SDP)** to study the transition between disordered (random) and ordered (flocking) phases. Using the **Processing IDE**, I extended the standard Boids flocking algorithm to observe how varying parameters like noise levels and particle density trigger a spontaneous symmetry breaking—a "novel" phase transition characteristic of active matter.

This implementation is based on the seminal research by **Tamás Vicsek et al.**

---

## 🚀 Key Features

* **Vector-Based Dynamics:** Written in Java using Processing’s `PVector` for efficient 2D/3D movement calculations.
* **Phase Analysis:** Real-time visualization of the order parameter (measuring global alignment).
* **Interactive Simulation:** Adjust noise, velocity, and interaction radius on the fly to observe state changes.

## 🛠 Technical Stack

* **Language:** Java
* **Environment:** [Processing 4](https://processing.org/)
* **Physics Model:** Vicsek Model / Reynolds Boids
